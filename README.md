Metro Munch

Metro Munch is a full-stack food ordering and management system built with Laravel, Livewire, Vite, MySQL, and Pusher. It allows restaurant staff to manage branches, food items, and orders in real time, with notifications for new orders.

Features

Multiple branches management

Add, edit, and delete food items with categories and variations

Live order notifications using Laravel Echo & Pusher

Customer management per branch

Order tracking and status updates
